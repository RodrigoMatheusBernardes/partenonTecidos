## 🎯 SEÇÃO 1: IDENTIDADE DA MARCA

### 1.1 QUEM É PARTHENON

**Nome Origin:**
- Parthenon = Templo grego dedicado a Atena
- Simbolismo: Estrutura, solidez, beleza eterna, sabedoria, craftsmanship divino

**Proposição de Valor:**
> "Tecidos que transcendem o ordinário. Cada fio conta uma história de excelência, seleção rigorosa e devoção ao ofício."

**Segmento:**
- Premium / Luxury
- Não é "barato" ou "rápido"
- É "Bom. Excelente. Eterno."

**Público:**
- Arquitetos / Designers de Interiores
- Confeccionistas Premium
- Pessoas que ENTENDEM tecido
- Que valorizam qualidade acima de preço

**Competidores Diretos:**
- Hermès (referência de excelência)
- Loro Piana (referência de sofisticação)
- Brunello Cucinelli (referência de elegância)

**Posicionamento:**
- NÃO somos "tecidos baratos"
- NÃO somos "tecidos rápidos"
- SOMOS "tecidos que você escolhe uma vez e leva para a vida toda"

### 3.3 TIPOGRAFIA

#### 3.3.1 FONTE PRINCIPAL: SERIF (Headlines)

**Escolha:** Playfair Display
```
- Pesos: Regular (400), Bold (700)
- Uso: Headlines, Títulos, Assinaturas
- Tamanho Mínimo: 24px (desktop), 18px (mobile)
- Espaçamento: 1.2 line-height para elegância
```

**Por quê Playfair?**
- Design inglês premium (desde 1897)
- Elegância clássica com sotaque moderno
- Nunca fica datado
- Comunica "isto é importante"

#### 3.3.2 FONTE SECUNDÁRIA: SANS-SERIF (Body)

**Escolha:** Inter
```
- Pesos: Regular (400), Medium (500), Semibold (600)
- Uso: Body text, Navegação, Labels
- Tamanho: 14px-16px (desktop), 13px-15px (mobile)
- Espaçamento: 1.6 line-height para leiturabilidade
```

**Por quê Inter?**
- Design feito para telas (2016)
- Legibilidade extrema
- Neutro mas com personalidade
- Funciona em qualquer tamanho

#### 3.3.3 HIERARQUIA TIPOGRÁFICA

```
H1 (Headlines principais)
- Playfair Display, Bold (700)
- 48px (desktop) / 32px (mobile)
- Line-height: 1.2
- Color: Terra Bruciata (#6B5344)

H2 (Subtítulos principais)
- Playfair Display, Regular (400)
- 36px (desktop) / 24px (mobile)
- Line-height: 1.3
- Color: Terra Bruciata (#6B5344)

H3 (Seções)
- Playfair Display, Regular (400)
- 28px (desktop) / 20px (mobile)
- Line-height: 1.4
- Color: Azul Cinza (#3D4E5C)

Body Text (Padrão)
- Inter, Regular (400)
- 16px (desktop) / 14px (mobile)
- Line-height: 1.6
- Color: #1A1A1A

Body Small (Secundário)
- Inter, Regular (400)
- 14px (desktop) / 12px (mobile)
- Line-height: 1.5
- Color: #9B9B9B
```

**Regra de Ouro:**
- Nunca mais de 3 tamanhos de fonte na mesma página
- Nunca misture Playfair com sans-serif no mesmo título
- Letras maiúsculas devem ser raras (menos impacto)

## 📐 SEÇÃO 4: SISTEMA DE ESPAÇAMENTO

### 4.1 ESCALA DE ESPAÇAMENTO (8px Base)

```
2px   = xs (micro ajustes)
4px   = xs (separação mínima)
8px   = sm (padrão)
12px  = sm-md (pequeno)
16px  = md (padrão médio)
24px  = md-lg (transição)
32px  = lg (padrão grande)
48px  = xl (espaço significativo)
64px  = 2xl (seções grandes)
80px  = 3xl (seções premium)
```

### 4.2 APLICAÇÕES

**Logo e Header:**
- Padding vertical: 24px
- Padding horizontal: 32px
- Espaço entre elementos: 16px

**Seções Principais:**
- Padding top/bottom: 80px (desktop), 48px (mobile)
- Padding left/right: 32px (desktop), 16px (mobile)

**Cards:**
- Padding interno: 24px
- Espaçamento entre cards: 24px (grid gap)

**Formulários:**
- Espaço entre campos: 16px
- Padding input: 12px (horizontal), 10px (vertical)

**Texto:**
- Espaço entre parágrafos: 16px
- Espaço após headline: 12px
